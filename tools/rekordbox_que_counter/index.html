How it works

Times use the mm.ss convention from the Excel sheet: 6.53 means 6 minutes 53 seconds.

Play = Cue out − Cue in (plain decimal subtraction, like column D).
Total = running sum of Play (column E).
Set time = Total read as minutes + hundredths-as-seconds, carried into h:mm:ss (column F).

Lists autosave to this device as you type. Save commits the list to your GitHub repo — the first save asks for a GitHub token, then locks it behind your password.
