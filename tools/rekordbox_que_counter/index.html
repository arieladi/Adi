How it works

Enter each track's Cue in and Cue out — the point where you bring a track in and out of the mix. Play Time = Cue out − Cue in, and Set time is the running length of the whole set.

Time formats: 5:18.6 (rekordbox, tenths), 5:18, 5.18 (5 min 18 s, like the old Excel), or 1:02:37.

▶ opens the track in a new tab — a YouTube search for the track name by default, or any link you set with ✎ (YouTube, SoundCloud, …). Rename the list with the ✎ next to its name.

Users: Public is open to everyone. Named users have a password and their own lists. Each user can keep up to 100 lists, 500 tracks per list.

Lists autosave to this device as you type. Save commits them to GitHub — one-time device setup with a GitHub token (⋯ → GitHub sync).
