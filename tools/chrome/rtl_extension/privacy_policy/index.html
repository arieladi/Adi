Privacy Policy

Extension: RTL/LTR Auto Direction
Developer: Adi Ariel
Last updated: June 2026

Overview

RTL/LTR Auto Direction is a Chrome extension that automatically switches text input direction between Right-to-Left (Hebrew, Arabic) and Left-to-Right based on the language being typed.

Data Collection

This extension does not collect, store, transmit, or share any data of any kind. Specifically:

  • No text you type is ever read, stored, or sent anywhere
  • No personal information is collected
  • No analytics or tracking of any kind
  • No network requests are made by this extension
  • No cookies are set or read

Permissions

The extension uses a content script that runs on all pages solely to detect the language of the first character typed on each line and adjust the text direction of the active input element. All processing happens locally in your browser. Nothing leaves your device.

Third Parties

This extension does not use any third-party services, SDKs, or APIs.

Changes

If this policy ever changes, the updated version will be published with the extension update.

Contact

For any questions, please open an issue on the Chrome Web Store listing.
